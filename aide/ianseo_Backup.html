Guide complet de sauvegarde d'Ianseo sous Windows

Procédure pour créer une sauvegarde complète de votre installation Ianseo

Ce guide reprend le principe du fichier initial fourni avec Ianseo, mais il inclut votre configuration, avec vos mises à jour et les modules additionnels installés.

Prérequis

1Télécharger et installer 7-Zip

7-Zip est un logiciel gratuit de compression de fichiers avec un excellent taux de compression.

Télécharger 7-Zip

2Vérifier si vous avez besoin de la version 32 ou 64 bits

Pour connaître l'architecture de votre système Windows :

  1. Ouvrez l'invite de commandes (cmd)
  2. Tapez la commande : msinfo32 ou systeminfo
  3. Recherchez la ligne Type du système
Vérifiez le "Type du système" (x64 ou x86)

Important : Une fois 7-Zip installé, vous avez tous les outils nécessaires pour effectuer une sauvegarde complète depuis zéro.

Procédure de sauvegarde

1Arrêter Apache et MySQL

Avant toute sauvegarde, il est crucial d'arrêter les services pour éviter la corruption des données.

  1. Ouvrez le panneau de contrôle XAMPP
  2. Cliquez sur Stop pour Apache
  3. Cliquez sur Stop pour MySQL
Arrêtez Apache et MySQL dans le panneau de contrôle XAMPP

2Fermer complètement XAMPP

Assurez-vous que XAMPP est complètement fermé, y compris l'icône dans la zone de notification.

Cliquez sur "Quit" dans XAMPP
Confirmez en cliquant sur "Quit"

3Exécuter 7-Zip en tant qu'administrateur

Pour éviter les problèmes de permissions, exécutez 7-Zip avec les privilèges d'administrateur.

Faites un clic droit sur 7-Zip et sélectionnez "Exécuter en tant qu'administrateur"

4Créer l'archive de sauvegarde

  1. Dans 7-Zip, sélectionnez le répertoire ianseo
  2. Cliquez sur le bouton Add to archive (Ajouter à l'archive)
  3. Sélectionnez le format 7z
  4. Cochez l'option Create SFX archive (créer une archive auto-extractible)
  5. Cliquez sur OK
Cliquez sur "Quit" dans XAMPP

Configuration de l'archive SFX dans 7-Zip

Note : L'option SFX crée un fichier .exe auto-extractible qui n'a pas besoin de 7-Zip pour être décompressé.

5Attendre la fin du processus

Patientez pendant que 7-Zip crée l'archive de sauvegarde. La durée dépend de la taille de votre installation Ianseo.

Une fois terminé, vous aurez un fichier .exe qui contient votre sauvegarde complète !!!

Restauration en cas de problème

Comment restaurer votre sauvegarde

Si vous rencontrez des problèmes avec Ianseo :

  1. Arrêtez Apache et MySQL comme indiqué précédemment
  2. Fermez complètement XAMPP
  3. Exécutez le fichier .exe de sauvegarde que vous avez créé
  4. Redémarrez XAMPP et les services

Vous retrouverez rapidement une installation Ianseo fonctionnelle !
